◆ Android ◆
1 · Install as an app
Adds an icon to your home screen. Recommended.
  1. Open this page in Chrome.
  2. Tap the ⋮ menu, top right.
  3. Tap Install app (or Add to Home screen).
  4. Confirm. The icon appears on your home screen.
Works offline after the first launch.
2 · Download the file
One HTML file on your phone. No app, no account.
  1. Tap Download HTML for offline use above.
  2. Accept the download. The file lands in Downloads.
  3. Tap the file and choose Chrome to open it.
Works with no connection at all, telescope WiFi included.
